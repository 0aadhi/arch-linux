# arch-linux


Sys 
Command line interface(cli) - arch linux
Window manager - dwm(dynamic window manager)
Terminal - st(suckless simple terminal)
App loader - dmenu
Status bar - slstatus


Using
Image viewer & wallpaper - feh
Transparency - picom
Video player - mpv
File manager - ranger,thunar
Document viewer - zathura 
Document editor - libreoffice
External drive Auto mount - udiskie
External drive Mount/unmount - udisks2
Audio - alsamixer(alsa-utils)
Wifi - network manager 
Bluetooth - bluez, bluezutils
Hardware statistics - btop
