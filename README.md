# arch-linux


Sys 
1. Command line interface(cli) - arch linux
2. Window manager - dwm(dynamic window manager)
3. Terminal - st(suckless simple terminal)
4. App loader - dmenu
5. Status bar - slstatus


Using
1. Image viewer & wallpaper - feh
2. Transparency - picom
3. Video player - mpv
4. File manager - ranger,thunar
5. Document viewer - zathura 
6. Document editor - libreoffice
7. External drive Auto mount - udiskie
External drive Mount/unmount - udisks2
Audio - alsamixer(alsa-utils)
Wifi - network manager 
Bluetooth - bluez, bluezutils
Hardware statistics - btop
